# QUICK START - SQUAD EDUCACIONAL

## TL;DR - Em 30 segundos

Squad Educacional = 4 agentes especializados + orquestrador central (Ralph) que resolvem problemas complexos em 4 passos estruturados com 90% de validação antes de código.

---

## OS 4 AGENTES (Funções Rápidas)

| Agente | O que faz | Pergunta Chave | Output |
|--------|-----------|-----------------|--------|
| **Discovery** | Clareza | "O que REALMENTE estamos resolvendo?" | Problema bem definido |
| **Analysis** | Decomposição | "Quais são as implicações?" | Compreensão profunda |
| **Architecture** | Design | "Como estruturamos a solução?" | Blueprint implementável |
| **Validation** | Verificação | "Está tudo correto e coerente?" | Confiança |

---

## OS 4 PASSOS (Flow)

```
1. CLAREZA DE DIREÇÃO
   └─ Entender o problema real

2. VALIDAÇÃO DE CONCEITOS (90% aqui!)
   └─ Debater entre 4 perspectivas

3. ESTRUTURAÇÃO LÓGICA
   └─ Desenhar a solução

4. PROTOTIPAGEM & ITERAÇÃO
   └─ Construir e aprender
```

---

## RALPH LOOP (O Maestro)

Ralph é o orquestrador que:
- Recebe INPUT (problema)
- Coordena os 4 agentes na sequência certa
- Gerencia consenso entre eles
- Produz OUTPUT (decisão + plano)

---

## TECH STACK (Ferramentas)

- **Claude** → Análise e decisão
- **Ralph** → Orquestração de agentes
- **Cursor** → Implementação prática
- **RepoMirror** → Análise de código existente

## DECISÕES-CHAVE (Por Quê)

| Decisão | Alternativa | Por Quê |
|---------|------------|---------|
| Squad | Assistente único | Múltiplas perspectivas > viés |
| 90% Validação | Code-first | 1h validação = 10h refator economizada |
| Framework genérico | Solução específica | Reutilizável para classes de problemas |
| Stack complementário | Ferramenta única | Especialização > generalismo |

## CHEAT SHEET - QUANDO USAR CADA AGENTE

**DISCOVERY:** Problema não está claro
→ Resposta: "O problema real é..."

**ANALYSIS:** Precisa entender implicações
→ Resposta: "Se fizermos isso, então..."

**ARCHITECTURE:** Precisa desenhar solução
→ Resposta: "A estrutura é..."

**VALIDATION:** Precisa validar coerência
→ Resposta: "Isso está correto porque..."
